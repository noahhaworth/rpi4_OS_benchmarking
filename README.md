# rpi4_OS_benchmarking
The following OSs were installed on a Raspberry Pi 4
* Alpine
* Arch Linux ARM
* DietPi
* Fedora 35
* FreeBSD
* Kali
* Kali (32 bit)
* Manjaro
* Raspberry Pi OS (new Raspbian)
* Ubuntu 20.04 LTS
* Ubuntu 21.10
* Ubuntu 21.10 (32 bit)
* openSUSE Tumbleweed

#Results
Rank | OS | Net runtime | Pvalue
|:---:|:---:|:---:|:---:|
1 | Rasberry Pi OS | 156.54 | 0
2 | DietPi (1.8 GHz) | 158.60 | 0
3 | FreeBSD | 163.77 | 0
