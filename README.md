# rpi4_OS_benchmarking

* Alpine
* Arch Linux ARM
* DietPi
* Fedora 35
* FreeBSD
* Kali
* Kali (32 bit)
* Manjaro
* Raspberry Pi OS (new Raspbian)
* Ubuntu 20.04 LTS
* Ubuntu 21.10
* Ubuntu 21.10 (32 bit)
* openSUSE Tumbleweed
